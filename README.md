# UnityPackageImporter
ロカールにあるunitypackageを一覧表示しインポートを可能とするエディタ拡張


Windowsのみ対応しています。  
NuGetでSharpZipLibをインストールしpackages/SharpZipLib.0.86.0/lib/20にあるICSharpCode.SharpZipLib.dllをPluginsに配置してください。
